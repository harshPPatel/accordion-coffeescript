  • Why to create Accordion with CoffeeScript 😲?

    This project is not a web app created with CoffeeScript ☕️, It just uses few lines of CoffeeScript to manipulate the DOM 🌲 of the website. Currently I am using CoffeeScript to toggle active class to each of these list items. I may change this code in teh future with class based CoffeeScript Code 🎭!
